# 本地 AI 聊天

这个目录现在是一个简化后的本地 AI 聊天界面，只保留最常用的功能：

- 聊天对话
- 连接测试
- 模型列表读取
- 文件上下文上传

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：前端逻辑
- `server.js`：本地代理服务，把 `/api/*` 转发到 `http://127.0.0.1:1234/*`

## 默认接口

页面默认请求：

`/api/v1/chat/completions`

它会通过本地代理转发到：

`http://127.0.0.1:1234/v1/chat/completions`

模型列表读取地址：

`/api/v1/models`

## 启动方式

在当前目录执行：

```powershell
node server.js
```

然后打开：

`http://127.0.0.1:8000`

## 当前界面保留的能力

### 1. 聊天

直接和本地模型对话，保留会话上下文。

### 2. 测试连接

点击“测试连接”可检查代理和本地 AI 服务是否可用。

### 3. 读取模型

点击“读取模型”会从 `/api/v1/models` 获取模型列表，并填充到下拉框。

### 4. 文件上下文

可以上传常见文本和代码文件，例如：

- `.txt`
- `.md`
- `.js` `.ts` `.tsx` `.jsx`
- `.json`
- `.html` `.css`
- `.java` `.py` `.go` `.rs`
- `.yml` `.yaml` `.xml`
- `.csv` `.sql`

当前限制：

- 单文件默认限制约 200 KB
- 注入到上下文的内容默认最多截取 12000 个字符
